# Operating Systems and Security Assignment

This repository contains four practical tasks implemented in C language on Linux.

## Tasks

### Task 1: Process Management and Threading
- Process creation
- POSIX threads
- Mutex synchronization
- Race condition prevention
- Round Robin scheduling
- Deadlock prevention

### Task 2: Memory Management Simulation
- Paging system
- FIFO page replacement
- LRU page replacement
- Page fault tracking
- Hit/miss ratio calculation

### Task 3: File System Operations and Security
- File create/read/write/delete
- User authentication
- File permission system
- Encryption and decryption
- Audit logging

### Task 4: Network Programming and IPC
- Client-server socket programming
- Simple communication protocol
- Multiple client handling
- Authentication and validation
- Error handling

## Language and Platform

- Language: C
- Platform: Linux
- Compiler: GCC
